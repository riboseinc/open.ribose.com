= 新功能：所有擎威 Ribose 平台電子郵件現在都使用 PGP 進行加密簽名

隨著電子郵件釣魚和欺騙行為日益普遍，許多人陷入了錯誤的絕望感。然而，有一些方法可以對抗這些行為，其中最常用的方法就是對電子郵件進行加密簽名。

自上週起，Ribose 已經開始對所有發送給您的電子郵件進行 PGP 加密簽名，以便您可以驗證我們電子郵件的真實性。有效的 Ribose PGP 簽名證明電子郵件在傳輸過程中未被修改，並且證明發件人確實是 Ribose，而不是惡意的第三方。

要驗證電子郵件的 PGP 簽名，目前需要安裝某種插件，但大多數這些工具都能很好地與您的平台（Linux / macOS / Windows）集成。我們還在下面的 PGP 頁面上列出了推薦的工具。

隨著我們的許多用戶來自敏感行業，其中一些人對加密電子郵件通知表示關注，以防止電子郵件中的數據洩露。作為下一步，我們將在即將發布的版本中啟用電子郵件的全面加密，並提供上傳您的公鑰 PGP 的功能，該公鑰將用於加密所有通過電子郵件發送的消息。敬請期待！

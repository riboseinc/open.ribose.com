= 擎威 Ribose 平台2013 年 7 月版本更新

本月初，我們釋出了最新的擎威 Ribose 平台更新。

最顯著的更新是全新的個人資料介面。我們使其更快速且更容易編輯和查看您的個人資料資訊。例如，您可以直接將滑鼠懸停並點擊您的姓名進行編輯。之前，個人資料僅顯示少數的聯絡人和空間，但現在我們顯示更多，讓您點擊更少。

在討論中，我們已將「評論」重新命名為「回覆」，以回應即將推出的功能。我們還修復了許多錯誤，涉及以下方面：
即將推出的功能，敬請期待。

其他變更：

* 聊天介面修復與聊天錯誤
* 日曆、檔案、空間選單、小工具、指導/幫助、密碼重設、電子郵件更改請求、邀請燈箱中的錯誤
* 通知小工具修復
* 維基：優化性能，修復錯誤

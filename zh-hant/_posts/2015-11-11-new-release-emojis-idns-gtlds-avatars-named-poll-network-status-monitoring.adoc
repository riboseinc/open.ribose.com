= 擎威 Ribose 最新版本發布：表情符號、IDN/gTLD 支持、頭像、命名投票、網絡狀態監控

大家好，

我們剛剛部署了最新版本，帶來了許多錯誤修復以及期待已久的功能，包括：

* 表情符號（以及完整的 Unicode）支持：適用於所有文本，無處不在。一張圖片勝過千言萬語！

* 支持 IDN 和 gTLD 電子郵件地址：之前不支持包含非 ASCII 字符和國際域名的電子郵件地址，如 .在线、.みんな、.شبكة 或 .сайт。現在它們都被視為一等公民——與 ASCII 電子郵件地址相同。同樣，我們現在支持帳戶電子郵件中的所有有效 gTLD。

* 頭像：用戶和空間的默認頭像現在使用他們名字的首字母。這使得用戶能夠在一定程度上輕鬆區分沒有個性化頭像的空間。當然，我們建議每個空間為用戶設置描述性的頭像！

* 命名投票：匿名和命名投票。之前只能發起匿名投票，我們收到了很多反饋，要求提供「命名投票」，允許在投票結果中顯示誰投了哪個選項。現在，當您在創建投票時選擇「投票匿名性 > 命名」時，就可以使用此功能。

* 網絡狀態監控：應用程序現在能夠在網絡離線時提供友好的提醒，告知用戶此時所做的更改將不會被保存。

* 安全加固和性能提升：本次版本的主要改進集中在安全性加固和性能提升方面。

如有任何問題、反饋或一般性意見，請隨時通過反饋表單或發送電子郵件至 support@ribose.com 與我們聯繫。

再次感謝您的支持！
